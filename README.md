# litecash-web-wallet
Web wallet interface for Litecash. Built with VueJS, requires Node (10+) to run.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
